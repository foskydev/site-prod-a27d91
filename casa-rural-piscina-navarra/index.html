Piscina privada para aprovechar la estancia

En verano, puentes y escapadas de fin de semana, una piscina privada cambia por completo la experiencia. Casa La Alcoholera no es un alojamiento compartido: se reserva de forma íntegra, con patio y zona exterior para vosotros.

Es una opción práctica para grupos pequeños y familias que quieren combinar visitas por Navarra con tiempo de descanso en la casa.

Un exterior para estar, no solo para mirar

El patio está pensado para hacer vida fuera: piscina, zona para comer, barbacoa y espacio para relajarse después de las excursiones.
